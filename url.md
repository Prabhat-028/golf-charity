# URL Reference

## Base URLs

- Local web app: `http://localhost:5173`
- Production web app: `https://your-domain.com` (replace with deployed domain)

## Public URLs

- `/` - Landing page
- `/login` - Login
- `/signup` - Signup
- `/signup/success` - Signup success
- `/signup/cancel` - Signup canceled
- `/forgot-password` - Forgot password
- `/reset-password` - Reset password
- `/terms` - Terms of Service
- `/privacy` - Privacy Policy

## User Panel URLs (Authenticated)

- `/dashboard` - User dashboard
- `/scores` - Submit/view user scores
- `/results` - Draw results list
- `/results/:id` - Result details
- `/wins` - Win history
- `/charities` - Charity information
- `/account` - Account/profile/subscription settings

## Admin Panel URLs (Admin Only)

- `/admin` - Admin dashboard
- `/admin/draws` - Draw management
- `/admin/users` - User management
- `/admin/winners` - Winner verification and processing
- `/admin/charities` - Charity allocation management
- `/admin/gdpr` - GDPR request management
- `/admin/settings` - Platform settings
