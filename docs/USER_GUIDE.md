# User Guide

## Getting Started

1. Create an account from the signup page.
2. Select monthly or yearly plan and complete Stripe checkout.
3. Log in and open the dashboard.

## Entering Scores

1. Go to **My Scores**.
2. Enter all 5 stableford scores.
3. Save scores to become eligible for the next draw.

## Checking Results

1. Open **Results** to view completed draws.
2. Select a draw to see winning numbers and prize tiers.
3. Open **Win History** to see your wins and payout status.

## Managing Account

In **Account Settings**, you can:
- Update your profile name and avatar
- Manage subscription in Stripe billing portal
- Update password
- Set email preferences
- Download your personal data export
- Submit a GDPR deletion request

## Admin Functions

Admins can:
- Manage draws
- Review users and winners
- Upload winner verification proof
- Manage charity entries
- Export CSV reports
- Process GDPR requests
